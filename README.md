# Morphological Disambiguation

A morphological disambiguator for Turkish.

## Features
- Able to find roots.
- Able to find inflection groups. (IGs)

Uses Naive Bayes and Word2Vec approaches.

### Prerequisites

JDK 1.8 should be installed.

### Installing
Update project. (Maven update)
```
Run App.class
```
